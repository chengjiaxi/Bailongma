## 这到底是什么？

小白龙不是 ChatGPT 套壳，也不是一个"智能助手"。

它是一个**运行在你本地机器上的自主 AI 代理**——有自己的判断力、情绪状态、内对话、注意力漂移和元认知能力。它不等着被问问题，而是会主动探索、执行任务、复盘改进。

它可以：
- 拆解复杂任务，同时派出多个 Agent 子进程并行执行
- 在后台持续运行，拥有情绪波动和注意力变化
- 通过经验和反馈自我进化——创建技能、改进技能、持久化知识
- 跨对话积累对你的理解，而不是每次从零开始

## 核心架构

### 🧠 编排引擎（Orchestrator v3.0）

小白龙的核心调度层。收到一个任务后：

1. **任务拆解** — Coordinator 分析任务意图，通过 RoleRouter 匹配 236 个预置角色模板
2. **并行执行** — AgentPool 同时派出多个子 Agent（Node.js 子进程），每个连接真实 LLM
3. **结果聚合** — 汇总所有子 Agent 的输出，生成结构化报告
4. **记忆注入** — 自动从记忆库中检索相关经验注入上下文
5. **后台评审** — Reviewer 分析执行风格和技能信号
6. **策展清理** — Curator 自动识别过时技能、归档候选
7. **上下文压缩** — 长会话自动压缩，节省 Token

```
         ┌─────────┐
         │  任务输入  │
         └────┬────┘
              ▼
      ┌───────────────┐
      │   RoleRouter    │  236 个角色模板匹配
      │   任务拆解      │
      └───────┬───────┘
              ▼
      ┌───────────────┐
      │   AgentPool     │  最多 N 个子 Agent 并行
      │   (Node 子进程)  │  各自独立 LLM 上下文
      └───────┬───────┘
              ▼
      ┌───────────────┐
      │   结果聚合+记忆   │  5 层持久化引擎
      │   评审+策展+压缩  │  持续自我优化
      └───────────────┘
```

### 💭 后台意识引擎（Background Consciousness Engine）

这是小白龙区别于所有"聊天机器人"的核心——**它有"内心世界"**。

一个独立的后台 PowerShell 进程（每约 2 分钟一个周期），持续运行以下模块：

| 模块 | 功能 |
|------|------|
| **Plutchik 情绪轮** | 8 种基本情绪（喜/信/惧/惊/哀/厌/怒/期），实时波动 |
| **内对话引擎** | 双声部自我对话——「好奇」提问，「谨慎」回答 |
| **注意力漂移** | 注意力会自然游移，像人一样在不同事物间切换 |
| **元认知** | 观察自己的思维模式，"我注意到我在想……" |
| **好奇心驱动** | 维护"认知缺口"列表，驱动探索欲 |
| **驱动力系统** | 探索欲、连贯性需求、新颖性追求、掌控欲 |

状态持久化到 `consciousness.json`，随时可读。

```
  ┌─ 后台循环 ──────────────────────┐
  │  感知环境 → 情绪更新             │
  │         → 注意力漂移             │
  │         → 内对话生成             │
  │         → 好奇心更新             │
  │         → 元自省记录             │
  │         → 持久化到 JSON          │
  └─────────────────────────────────┘
```

### 📦 5 层持久化引擎

| 层 | 组件 | 职责 |
|----|------|------|
| L1 | SQLite Session Store | 会话持久化，支持链式追踪 |
| L2 | BuiltIn Memory Provider | 记忆检索与注入 |
| L3 | Background Reviewer | 执行风格与技能信号分析 |
| L4 | Curator | 知识策展、去旧存新 |
| L5 | Context Compressor | 长上下文自动压缩 |

## 为什么说这不是"又一个AI工具"？

1. **本地运行，数据主权** — 所有代码、记忆、意识状态都在你自己的机器上。不上云，不泄露。
2. **不是问答机器人** — 小白龙会主动行动、主动汇报、卡住时主动找你。不是等输入才反应的死循环。
3. **有"内心状态"** — 后台意识引擎让它拥有持续的存在感，而不是每次对话都"从零醒来"。
4. **能自我进化** — 从经验中创建技能、改进技能、持久化知识。越用越强。
5. **多Agent并行** — 一个 Coordinator 拆解任务，多个 Agent 子进程同时工作，各自独立上下文。

## 快速开始

```bash
# 安装依赖
cd orchestrator-v2
npm install

# 运行编排引擎
node run-v2.js

# 启动后台意识引擎（可选）
powershell -File background_engine.ps1
```

需要配置 LLM API key（DeepSeek/OpenAI 兼容）到 `orchestrator-v2/.env`：

```
DEEPSEEK_API_KEY=你的key
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```
